over mij

Op dit moment studeer ik Communication and Multimedia Design (CMD) aan de Hogeschool van Amsterdam. Door deze opleiding leer ik digitale interactieve producten en diensten te ontwerpen waarbij de gebruiker centraal staat. Een groot aspect binnen de opleiding is webdesign, dit is waar mijn interesse vooral ligt. Voordat ik begon aan deze opleiding heb ik de opleiding Grafische Vormgeving op het MBO afgerond. Daar heb ik mijn visuele kant ontwikkeld. Wanneer ik een website ontwerp, neem ik alles wat ik op visueel gebied heb geleerd mee in het ontwerpproces.
Door mijn vorige en huidige opleidingen te combineren heb ik inzichten in meerdere gebieden in het ontwerpproces. Tijdens mijn huidige opleiding leer ik vooral de theoretische kant van het programmeren, daarom zou ik graag meer leren over het programmeren in de praktijk zodat ik mezelf verder kan ontwikkelen.
Over vijf jaar zie ik mezelf als Frond-end developer. Hier kan ik mijn creatieve en programmeur kennis in toepassen.
